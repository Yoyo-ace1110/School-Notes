## 簡單式 vs 完成式 vs 完成進行式
- 1. John *lived* in Paris for five years. (過去簡單式)
- 2. John *has lived* in Paris for five years. (現在完成式)
- 3. John *has been living* in Paris for five years. (現在完成進行式)
- ### 總結對比表
    | 句子 | 時態 | John 現在在哪？ | 語意側重點 |
    | :--- | :--- | :--- | :--- |
    | **Lived** | 過去簡單式 | 不在巴黎 | 一段**已結束**的往事 |
    | **Has lived** | 現在完成式 | 還在巴黎 | 累積至今的**狀態** |
    | **Has been living** | 現在完成進行式 | 還在巴黎 | 這段時間**持續不斷**的過程 |

## **縮寫完整對照表**

### 1. 代名詞與助動詞/連綴動詞 (Pronoun + Verb)
這類縮寫是口語中最核心的部分，撇號通常取代了動詞開頭的 1~2 個字母。
| 縮寫形式 | 代表單詞 | 範例 | 備註 |
| :--- | :--- | :--- | :--- |
| **'m** | **am** | I**'m** | 僅限於 I。 |
| **'re** | **are** | You**'re**, We**'re**, They**'re** | 注意不要跟 Your 混淆。 |
| **'s** | **is** / **has** | He**'s**, She**'s**, It**'s** | 後接 V-ing/Adj 為 **is**；後接 V-pp 為 **has**。 |
| **'ve** | **have** | I**'ve**, You**'ve**, We**'ve**, They**'ve** | 表示「完成」或「擁有」。 |
| **'ll** | **will** | I**'ll**, You**'ll**, He**'ll**, We**'ll**, They**'ll** | 表示未來或意圖。 |
| **'d** | **had** / **would** | I**'d**, You**'d**, He**'d**, She**'d**, We**'d** | 後接 V-pp 為 **had**；後接原形 V 為 **would**。 |

### 2. 動詞與否定詞 (Verb + Not)
這類縮寫撇號取代了 **not** 中的 **o**。
| 縮寫形式 | 代表單詞 | 範例 | 特殊變體 |
| :--- | :--- | :--- | :--- |
| **n't** | **not** | Is**n't**, Are**n't**, Has**n't**, Have**n't** | 最通用的否定縮寫。 |
| **-** | **can + not** | **Can't** | 省略了 "no"。 |
| **-** | **will + not** | **Won't** | 拼寫產生劇烈變化，需特別記住。 |
| **-** | **shall + not** | **Shan't** | 現代英文較少見，多見於英式或文學。 |
| **-** | **do / did + not** | **Don't**, **Doesn't**, **Didn't** | 助動詞否定。 |

### 3. 疑問詞與動詞 (Question Word + Verb)
這類縮寫主要出現在非正式口語中，寫作時通常建議寫全名。
| 縮寫形式 | 代表單詞 | 範例 | 備註 |
| :--- | :--- | :--- | :--- |
| **Who's** | **Who is / has** | **Who's** there? | 常與所有格 Whose 混淆。 |
| **What's** | **What is / has** | **What's** up? | 表示「是什麼」或「發生了什麼」。 |
| **Where's** | **Where is / has** | **Where's** my key? | 地點查詢。 |
| **How's** | **How is / has** | **How's** it going? | 狀態查詢。 |
| **When's** | **When is / has** | **When's** the party? | 時間查詢。 |

### 4. 特殊、口語與時間縮寫 (Special & Slang)
這些縮寫不一定遵循固定的語法規律，多為習慣用法。
| 縮寫形式 | 完整單詞 | 範例 | 語境 |
| :--- | :--- | :--- | :--- |
| **Let's** | **Let us** | **Let's** go! | 提議、祈使句。 |
| **o'clock** | **of the clock** | Ten **o'clock** | 時間表示法。 |
| **y'all** | **you all** | How **y'all** doin'? | 美國南方口語（複數的「你們」）。 |
| **'cause** | **because** | Just **'cause** I can. | 非正式、對話中常用。 |
| **ma'am** | **madam** | Yes, **ma'am**. | 尊稱（女士）。 |
| **'em** | **them** | Tell **'em** the truth. | 口語中省略 th。 |
