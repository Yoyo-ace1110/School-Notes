# CH4 三角比全

- ### 角平分線長度
  - 令角A為 $2\theta$, 角平分線長為 x
  - $x = 2cos\theta (\frac{bc}{b+c})$
- ### 中線長度
  - 令BC邊上的中線長為 x
  - $x = \frac{1}{2}\sqrt{2b^2+2c^2-a^2}$
- ### 折線長度(平行四邊形)
  - 令對角線長為 x
  - $摺痕長 = x \cdot tan \theta$
- ### 海龍公式
  - $S_{\triangle ABC} = \frac{1}{4} \sqrt{4a^2b^2-(a^2+b^2-c^2)^2}$
