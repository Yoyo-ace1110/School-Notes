# 地形系統與營力

## 等高線地形圖
  - #### 計曲線
    - 每隔四條會有一條(約佔 **$\frac{1}{5}$**)
    - **粗線**，標明**高度值**
  - #### 首曲線
    - 又稱主曲線
    - 最多(約佔 **$\frac{4}{5}$**)
    - **細線**，不標明**高度值**
  - #### 助曲線
    - 畫在首曲線和計曲線之間
    - 為**細虛線**
    - 用於表示細部地形
    - 👉 平原兩條首曲線距離遠 -> 補上助曲線
  - #### 示坡線
    - 表等高線趨勢與本來不同
    - 又難以表達出來
    - 向內長毛
    - 👉 火山湖 ![示意圖](./asset/4-1-1.jpg)
  - #### 判讀
    - V指向低處 -> 脊(較高)
    - V指向高處 -> 谷(較低)
    - #### 鞍部(+處)
      - 低中低   低中低
      - 中高中 + 中高中
      - 低中低   低中低

## 地形三要素
  - ### 1. 構造
    - #### 岩石種類
      | 岩石種類 | 舉例(...岩) | 特性 |
      | :--: | :-- | :--: |
      | 火成岩 | 花崗->安山->玄武 | 冷卻越快;顆粒越小 |
      |        | 橄欖岩...       | 侵入岩顆粒大;噴出岩顆粒小 |
      | 沉積岩 | 礫岩->砂岩->頁岩 | 地表主成分，風化+膠結而成 |
      |        | 石灰岩... |
      | 變質岩 | 板岩 片岩 片麻岩 石英 大理岩 | 高溫高壓形成 |
    - #### 差異侵蝕
      - 👉 蕈岩: 像是女王頭
      - 👉 風化窗: 地表呈網格狀
    - #### 岩層排列
      - ![示意圖](./asset/4-1-2.jpg)
    - #### 惡地地形
      - 沉積在淺海底的泥土，在尚未膠結完全之前就被抬升出露地表
      - 透水性低的泥岩難以讓植物生長，遇雨易被沖刷出一條條雨溝
      - 👉 高雄月世界(荒涼)
  - ### 2. 營力
    - ### 內營力
      - 海vs海 or 陸vs海 -> 海溝、島湖
      - 陸vs陸 -> 喜馬拉雅山
      - 向斜易有地下水;背斜易有油氣
      - 下陷 -> 地塹 ; 上升 -> 地壘
      - #### 火山地形...
    - ### 外營利
      - 搬運能力 -> 淘選度: 顆粒的**大小均勻程度**
      - 侵蝕能力: 冰河 > 海水 > 雨水 > 河(冰)水 > 風 ?
      - 淘選度: 風 > 河(冰)水 > 海水 > 冰河
      - #### 風化
        - 物理風化(較弱): **鱗剝、凍融**作用 
        - 化學風化(較強): 風化窗、石灰岩地形
        - 生物風化: 樹根生長...
