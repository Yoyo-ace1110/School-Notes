# 河流地形(主)

## 河流地形
  - ### 河水作用
    - #### 侵蝕
      - 向**源**侵蝕
      - 向**下**侵蝕
      - 向**兩側**侵蝕
  - ### 地形發育過程
    - #### 幼年期
      - 高山深谷，坡度陡
      - 侵蝕 > 堆積
    - #### 壯年期
      - 稍微平緩
      - 侵蝕 ≈ 堆積
    - #### 老年期
      - 準平原
      - 侵蝕 < 堆積
    - #### 襲奪作用
      - 高位河側蝕切穿分水嶺
      - 連到低位河(搶水河)
  - ### 曲流
    - 岩石硬度差異(或是其他原因) -> 水流向一邊侵蝕
    - 侵蝕處成為**凹岸** ; 堆積處成為**凸岸**
    - 越來越**彎** -> 凸岸連接處越來越**細**
    - 形成**曲流頸** -> 河水改道 -> **牛軛湖**
    - 河水改道時**環繞**原本的凸岸 -> 形成離堆丘
    - 同時形成河階 -> 劇場河階

## 海岸地形
  - #### 海水運動
    - 波浪: 對海岸侵蝕力最大
    - 潮汐: 河口堆積
    - 海流: 洋流、沿岸流
  - ### 海岸分帶
    - #### 海岸線
      - 碰不到水，海水作用終點
    - #### 浪裂線
      - 海邊白色浪花線
    - #### 濱線
      - 潮間帶中點
      - 前濱: 潮間帶之間
      - 後濱: 高潮位線~海岸線
  - ### 海積地形
    - #### 濱外沙洲
      - 濱線外堆積 -> 沙洲
      - 易形成暗灘(後方堆積)
    - #### 沙嘴
      - 沙洲一側與陸地相連
      - 封閉掉溪流 -> 沒口溪
    - #### 海灘
      - (顆粒大) 礫灘 -> 沙灘 -> 泥灘 (顆粒小)
      - 可能形成海岸沙丘
    - #### 連島沙洲
      - 島嶼往路地堆積
    - #### 陸連島
      - 因沙洲與陸地相連
    - #### 沙頸岬
      - 連島沙洲 + 陸連島


## 地形演育
  - ### 石灰岩地形
    - 滲穴(小凹洞) -> 漥盆(凹洞變大)
    - 漥盆 -> 錐丘(凹洞大，剩下的看似凸出)
    - 錐丘 -> 殘丘(沒那麼尖，快沒了)
  - ### 海蝕地形1
    - 海蝕溝or海蝕洞 -> 海蝕凹壁(連片)
    - 海蝕凹壁 -> 海蝕崖(上方崩塌) 
    - 海蝕崖 -> 海階(抬升出水面)
  - ### 海蝕地形2
    - 海蝕洞 -> 海蝕門(變大)
    - 海蝕門 -> 海蝕柱(崩塌)
  - ### 河流地形
    - 幼年期 -> 壯年期 -> 老年期
