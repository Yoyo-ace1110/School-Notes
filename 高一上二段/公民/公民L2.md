# L2 國家主權與認同

### 國家認同(簡略)
#### `def`: 
1. 對國家的歸屬感
2. 願意為共同生活努力，甚至在危難時願意犧牲付出
#### 三種原因: 
1. 血緣種族
2. 歷史文化
3. 社會體制
4. **公民身分保障**

### 我國對少數族群的具體措施
保障少數族群的公民權
1. 身心障礙者: "殘障"福利法 -> 身心障礙者權益保障法  
   被保護的殘障 -> 擁有平權的公民
2. !異性戀者: 民法、司法院釋字第748號解釋施行法(同婚專法)
   東亞第一可同婚的國家
3. 原住民: 原住民族基本法、原住民身分法、原住民族教育法、原住民族工作權保障法

### 國家主權與領土
擁有者: 全體國民(多數民主國家)  

四種主權:
1. 對內主權: 對內的最高性
2. 對外主權: 對外的獨立性
3. 合法主權: 法律上聲稱，且獲國際上承認
4. 事實主權: 實際管轄範圍(與法律無關)

主權範圍:

### 我國的主權與領土
- 憲法1: 中華民國基於三民主義，為民有民治民享之民主共和國。  
- 憲法2: 中華民國之主權屬於國民全體。  
- 憲法4: 中華民國領土，依其固有之疆域，非經國民大會之決議，不得變更之。  

目前將中華民國分為兩地區，分裂分治:
    1. 自由地區: 台澎金馬
    2. 大陸地區: other
