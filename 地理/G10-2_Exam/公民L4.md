# L4 公民參與與民主治理

## 選舉制度

- ## 多數決制
  - 得票較多的人贏
  - ### 相對多數決制
    - ✅得票為相對多數者贏
    - ❌若得票率沒有過半，會被認為正當性不足
    - #### **單一選區相對多數決制**
      - 台灣選總統、市長、區域立委的方式
      - 得票最多的**一個人**當選
      - ✅規則很簡單
      - ❌贏者全拿，對**大黨**有利
    - #### **複數選區相對多數決制**
      - 台灣選議員、原住民立委的方式 
      - (女性 >= n//4)
      - 依得票數排下來，前**n**名當選
      - ✅有一定比例就可能當選，對**小黨**有利
      - ❌可能會出現較激進的政黨/無黨籍候選人
  - ### 絕對多數決制
    - 當選者最後(得票率>50%)
    - 先選第一輪，若有(候選人得票率>50%)則當選
    - 若沒有超過50%，則讓得票數前2名進入第二輪
    - 第二輪得票多者當選
    - ✅勝負明顯，爭議較少
    - ❌很有可能需要兩輪投票，勞民傷財

- ## 比例代表制
  - 我國選不分區立委的方式
  - 將政黨得票率轉為候選人席次
  - 有可能設有門檻，小於門檻的政黨無法獲得席次

- ## 混合投票制
  - 結合前兩者(多數決制 && 比例代表制)
  - 通常會有兩張選票，一張選人一張選黨
  - 由於有兩張選票，又稱作單一選區兩票制
  - ### 並立制
    - 我國選立委(整體)的方式
    - 不分區立委（女性 >= 1/2)
    - 政黨總席次 = 區域席次 + 政黨得票率*不分區立委總席次
    - ✅名額固定，區區都有代表，不會小黨林立
    - ❌得票率沒有直接對應到席次，對大黨有利
  - ### 聯立制
    - 德國選聯邦議員的方式
    - 政黨總席次 = 總席次*得票率
    - 不分區立委總席次 = 政黨總席次 - 區域席次
    - 若(區域席次 > 政黨總席次)則會超額
    - #### 超額的處理方式
      - 德國以前 -> 直接超收1立委 -> 一堆超額
      - 德國現在 -> 砍掉區域立委 -> 沒有代表
    - ✅得票率直接對應到席次，有等比例性
    - ❌會超額、小黨林立 -> 在內閣制國家容易要組聯合內閣

> 傑利蠑螈現象: 透過不當的選區劃分，以獲得更多席次
> 利益團體又稱壓力團體

## 罷免
1. ### 正副總統  
    - 需就職滿一年
    - 沒有被罷免失敗過
    - #### 流程:
      1. **¼** 以上立委提議
      2. 全體立委 **⅔** 以上同意
      3. 公告投票
      4. 選舉人總額過半數投票 && 有效同意票數多於不同意票數
      5. 立即解除職務
2. ### 一般公務員
    - 需就職滿一年
    - (不分區、僑選)立委不適用
    - #### 流程:
      1. 原選舉區1%以上公民提案
      2. 原選舉區10%以上公民連署
      3. 公告投票
      4. 25%以上公民同意 && (同意 > 不同意)

## 公投
   - #### 預算, 租稅, 薪俸, 人事不適用
   - ### 提案
     - #### 行政院 && 立法院
       - 內容: 重大政策的**創制或複決**
       - 門檻: 立法院決議
     - #### 總統
       - 內容: 國安事項
       - 門檻: 行政院決議
     - #### 人民
       - 內容: 
         - **立法原則**的創制
         - 法案的複決
         - 重大政策的創制或複決
       - 門檻: 
         - 提案人數: 前次總統大選的 $\frac{1}{10000}$
         - 連署人數: 前次總統大選的 1.5%
   - ### 流程
     - 公告至少90天
     - 資格: 國民, 18歲, 住6個月
     - 門檻: (同意 > 不同意) && (同意 > $\frac{all}{4}$)

### 公私協力
   - 一種夥伴關係
   - 第1部門(公部門)  : gov
   - 第2部門(私營部門): 企業 
   - 第3部門(志願部門): 非營利組織 or 民間團體 

\# 民主治理 -> 雙向平等  
\# 專家治理 -> 專家會議, 公聽會, 書面研究報告
